This is an interstitial

If you need to give the user a temporary message that will go away, use this. This message will go away after 15 seconds. Using a loader/spinner here is also a good idea. Having a cancel or alternative CTA is smart here as well.

Basic message modal

This modal would be used if there is an imperative message that is required to be seen by the user. It should be revealed based on a users action only. Ex. Displaying an alert because a user made an error and cannot 'proceed'. More than likely we would use a CTA.

Basic message modal (smaller)

This is the same as the basic modal except a little smaller. Just add a class="smaller" and you can keep the size down if you have less content.
