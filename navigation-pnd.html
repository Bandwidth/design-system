Navigation

Pages have been rearranged or combined to form this new structure.

PRIMARY & SECONDARY

  • Numbers includes:

    • My Numbers, a place to view your Bandwidth network numbers and external numbers, add or remove external numbers, and configure line features for one or many numbers
    • Buy, a place to order new numbers, import a batch from a spreadsheet, or access past new number orders
    • Port, a place to check portability, port numbers onto or off (LSR) the Bandwidth network, or access past port requests
    • Disconnect, a place to disconnect one or many numbers as you can do today, and access to past disconnect orders
    • Transaction History, a collection of all past "orders", separated by order type
    • Groups, a place to add, edit, or delete sub-accounts (aka "sites", i.e. address groups) and locations (aka "SIP peers", i.e. IP groups)

  • Applications includes:

    • A single page to add, edit, or delete both voice and messaging applications as they exist today in Catapult

  • 9-1-1 includes Dash stuff (what exactly is that?)

  • Reports includes:

    • the same "Reports Dashboard" that exists today. "Billing Detail Records" will be moved under Account > Billing.

UTILITY

  • Search: a link that will reveal a full-page search experience where you can type in phone numbers, order IDs, sub-account or location names, etc. Phone number searches will bring the user to existing details pages for numbers they own or suggest it's available for purchase, porting, etc.

  • Documentation: an external link to API documentation

  • Support:

    • Short term - an external link to Bandwidth Support (Zendesk)
    • Long term - opens a panel or full-page experience with ways to read FAQs, watch learning lab videos, live chat, etc.

  • Account includes:

    • Account Overview,
    • Users, the new user management experience
    • Notifications
      • Short term - an updated version of the existing experience
      • Long term - this could be moved to a new box within individual user profiles
    • Billing, an updated experience in the works that will contain invoices, CDRs (currently "BDR"), MDRs, Statement BDRs, DID snapshots (for DTH, or Phone Number History for TRACT), and credit card payment
    • Trunking Settings, same as today, needs UX exploration
    • SIP Credentials, the new experience that's near completion

  • Log out
