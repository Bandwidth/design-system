For those email clients that don't support media queries, like Gmail, each block will maintain its set width, so be careful using smaller blocks for wide content.

Hybrid cells

These fixed-width blocks will stack when there's not enough horizontal space. Because they're pixel-based instead of percentage-based, the fluidity of their size is limited to certain minimum and maximum widths. Because of this stacking for mobile devices, they've all be set to center alignment. CSS media queries are then added as a progressive enhancement for email clients that support it.
